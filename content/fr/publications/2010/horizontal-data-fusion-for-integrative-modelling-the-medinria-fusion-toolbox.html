Through the use of a core/plug-ins architecture, it explores solutions to make it easy for researchers to import their own tools, wrap them around a single application or exchange them between applications supporting the same framework. That attempt at a universal solution is a step towards standardisation of the exchange of algorithms. Standardisation calls for sustainability, maintainability. The proposed framework can also help quantitative and qualitative tests of functionally similar methods. Medinria-fusion is a demonstrator that implements a registration algorithm, using the API we propose as an open framework. This software allows to visualise several formats of images (among them is DICOM). citation_full: >- Benoît Bleuzé, Olivier Clatz, Pierre Fillard, Maxime Sermesant, Nicolas Toussaint, et al..
